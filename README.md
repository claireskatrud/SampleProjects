**2. New Commons Submission Analysis for The Governance Lab Summer 2025**
I was a data program intern for the GovLab, a research organization based at New York University. This pdf is an informal, internal report that summarized my analysis of project proposals for the GovLab's New Commons Challenge. The GovLab's Open Data Policy Lab hosted the New Commons Challenge and received over 150 funding proposals for projects focused on creating data commons, which are collectively governed data ecosystems. I performed this analysis primarily with Excel. I used Python for certain processes like parsing the "Address" field of the submissions using the Google Maps API to identify the city and country. Because this analysis was entirely internal, this analysis assumes background information that a general audience might not have.
To read more about the New Commons Challenge, you can check out this blog post that I wrote for the GovLab's website: https://opendatapolicylab.org/articles/new-commons-phase-one-stats/
